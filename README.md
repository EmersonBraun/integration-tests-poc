## Install

```sh
npm ci --silent
```

## Usage

```sh
npm run test
```

## Test
```bash
# tests
$ npm run test

# tests watch
$ npm run test:watch

# test coverage
$ npm run test:coverage
```

## Git Hooks
Adding a hook:
```bash
npm husky add .husky/pre-commit "npm test:cov"
```
More: [Husky Official Page](https://typicode.github.io/husky/)
